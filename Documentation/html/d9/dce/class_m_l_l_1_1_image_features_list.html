Constructor that will extract a list of feature vectors for all the image files in the specified directory.

Will scan the directory _dirName for any image files. For each image found a new instance of ImageFeatures will be created who's features will be derived from the image. These ImageFeatures' objects will be assigned the class specified by '_mlClass'. A new data file containing the extracted features will be saved in fileName.

Parameters
[in]_logLog file to write messages to.
[in]_mlClassClass to assign to new 'ImageFeatures' objects.
[in]_dirNameDirectory to scan for images.
[in]_fileNameName of file to contain the extracted feature data. Will be og the Raw format.

Calculates the position in meters of each plankton Feature-Vector with respect to the first plankton particle in the SIPPER file.

It is assumed that all the items in this list are from the same SIPPER file. Item(FeatureVector) will be assigned a position in meters with respect to the first particle in the file.

Parameters
[in]instrumentDataProvides the flow rate from the instrument data will be used to calc distance traveled; otherwise 'defaultFlowRate' will be used.
[in]defaultScanRateScan lines per second that the camera operates at, if <= 0 will assume 24950.0 scan lines/sec.
[in]defaultFlowRateFlow rate to use if none provided by 'instrumentData'. if <= 0.0 will default to 1.0 meters/sec.
Returns
A Vector of doubles where each element in it will represent the distance in meters from the first particle in the SIPPER file. There will be one entry for each "ImageFeatures" object in the list and it will be sorted from low to high.
