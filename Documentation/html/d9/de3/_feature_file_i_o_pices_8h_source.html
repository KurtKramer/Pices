20  /**
21  @class FeatureFileIOPices
22  @brief Supports the reading and writing of feature data from and to Pices feature data files.
23  @details
24  @code
25  * *************************************************************************************************
26  * * The PICES feature data file is a very simple format where the first few rows contain a *
27  * * description of the rest of the data file. Any row that starts with "//" will be treated as *
28  * * comment line. *
29  * * First Row: Identifies the file as a feature Data File. An example on the next line. *
30  * * "FEATURE_DATA_FILE Version 51 NumOfFeatures 88 NumOfImages 258" *
31  * * Each field will be separated by a tab character. This line says that the *
32  * * version number is 51. There are 88 features and 258 examples or feature vectors. *
33  * * The version number gets incremented when ever there is a change in the way *
34  * * that feature data is computed. *
35  * * *
36  * * Second Row: List all the fields in the feature data file. The first fields are feature *
37  * * data. The first non feature field will be: "ClassName". There are several no *
38  * * feature data fields. See the SipperSoftware documentation for details. *
39  * *************************************************************************************************
40  @endcode
41  @see FeatureFileIO
42  */
