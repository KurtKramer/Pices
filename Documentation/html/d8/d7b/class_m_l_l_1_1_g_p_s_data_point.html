Detailed Description

Represents a single GPS data-point entry (GMT DateTime, Latitude and Longitude) from a imported GPS data source provided by the research Vessels that SIPPER is deployed on.

Research vessels will record GPS data into text files throughout the length of their cruises. Temporal granularity will range from a couple seconds to a minute. This data is imported into the PICES database via the ImportGPSDataApp application. The data is kept in the GpsData table of the PICES database.
