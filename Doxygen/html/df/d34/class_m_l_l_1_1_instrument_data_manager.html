Detailed Description

Manages instrumentation data supplied by SipperBuff derived classesforwarding to appropriate 'InstrumentDataReport' derived objects.

Responsible for managing instrument data that is collected by SipperBuff or one of its decedents. Originally developed when SIPPER3 format came out. This format includes instrument data. Will maintain a list of 'InstrumentDataReport' instances. One for each Instrument/Sensor and Data or Text. SipperBuff will call the 'ReportInstrumentData' method which will decide the correct 'InstrumentdataReport' instance to send it to, and if necessary, create a new instance of one.
