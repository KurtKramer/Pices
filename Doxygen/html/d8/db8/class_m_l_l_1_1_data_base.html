Returns a List of FeatureVector's for specified SipperFile and if specified for Specified ImaegClass.

Loads all Images with related Feature data for the specified 'SipperFile'.

The retrieval is driven by the Images table and a Left-Join is done with the FeatureData table; so if there is no corresponding FeatureData table entry for a image the FeaterData will be filled with 0's.

Parameters
[in]classKeyToUseIf equal to 'V' will only return Validated examples.
[in]reExtractInstrumentDataIf set to true will Re-Extract Instrument data from the SipperFile.
[in]sipperFileNameSipper file to retrieve feature data for.
[in]mlClassYou are only interested in feature data for a specific class; whether predicted or validated class depends on 'classKeyToUse'.
[in]classKeyToUseIndicates which ClassId field to use for selecting feature data by class; 'V' = 'Images.ClassValidatedId' otherwise 'Images.Class1Id'.
[in]reExtractInstrumentDataTrue indicates to re-extract feature data from the original SipperFile and update InmstrumentData tables.
[in]cancelFlagFlag will be monitored; if turns true the method will return at 1st chance.
