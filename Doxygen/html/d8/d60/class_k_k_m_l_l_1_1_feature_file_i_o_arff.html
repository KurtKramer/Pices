Detailed Description

Support the writing of ARFF Formatted Feature Files.

Supports the writing of the ARFF format as supported by the WEKA machine learning system The ARFF format is very similar to the C45 file format; but there is only one file that contains the file description and examples in it. see http://weka.wikispaces.com/ARFF+(book+version for description.
