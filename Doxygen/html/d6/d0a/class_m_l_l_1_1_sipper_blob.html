Detailed Description

Used by SipperImageExtrtaction to manage a single blob.

Author

SipperBlob *
*
Used by SipperImageExtraction to keep track of an individual connected *
components in a sipper frame. We call these connected components blobs.*
The fields rowTop, rowBot, colLeft, colRight indicate a tight bound- *
box around the blob. SipperImageExtraction will use *
