45  *@details This class is meant to be a general class that all standalone applications should be inherited
46  *from. If provides command line options that are common to all Pices Applications.
47  * Supported Command Options:
48  * -# -Config <Model-Name> The training model configuration file; only the root part needs to be specified; these files

50  * -# -DataBase <Description> This 'Description' references the Description field in the MySQL.cfg file found in
51  * "${PicesHomeDir}\Configurations\" directory.
52  * -# -LogFile <File-Name> File that logger messages (RunLog) are to be written to; if not specified will default
53  * to standard-out (cout).
54  * -# -CmdFile <File-Name> A text file that will contain additional command line options; one of these options could
55  * be another "-CmdFile" option.
56  *
57  * A derived class must implement the following methods:

59  * - InitalizeApplication Where you implement the initialization code for the derived application. One of the tasks required
60  * by this method is to call the base class version of this method "PicesApplication::InitalizeApplication".
61  * If you find ant reason that the application may not be started; display an appropriate error message and
