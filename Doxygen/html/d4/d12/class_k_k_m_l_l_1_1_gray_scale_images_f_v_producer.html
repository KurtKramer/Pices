Compute a FeatureVector for the supplied 'image'.

Parameters
[in]image
[in]knownClassThe class label that will be assigned to this Feature vector.
[in,out]intermediateImagesIf not NULL will save a copy of the intermediate images created during Feature Computation; the call will own the list and its contents.
[in]priorReductionFactorThe reduction that was done to the image prior to calling this routine; if no size change was made then set this field to 1.0. The FeaureVector computation routines will use this field to adjust SizeDependent features appropriately. the value of this parameter represents the number of pixels that were reduced to 1 pixel. A value of 3 indicates (3 x 3) pixels were reduced to 1 pixel.
[in]logLogging file.
