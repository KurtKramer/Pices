Detailed Description

A abstract class that is meant to compute a FeatureVector from a source image.

Applications that want to utilize this library will either use one of the provided "FeatureVectorProducer" derived classes or supply their own. This class will be responsible for computing a FeatureVector from a supplied Image.

Each FeatureVectorProducer derived class will need to have a unique name that will be supplied when constructed. This name will be used later when it is required to locate the appropriate FeatureVectorProducer to utilize.
