Detailed Description

The Managed version of the OSServices methods defined in BaseLibrary.

remarks> This class is the managed equivelent to the functions defined in OSservices in BaseLibrary. In some cases we implement the code in the unmanaged world in others we encapsulate the Unmanaged call. /remarks>
