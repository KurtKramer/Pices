A virtual Constructor for InstrumentDataReport instances.

Based on the parameters will instantiate an instance of the correct derived. In the future as different types of instruments are introduced we can make specialized versions of InstrumentDataReport.

Parameters
[in]_managerInstrumentDataManager instance that will this newly created instance will report to.
[in]_reportDirDirectory where report file is to be placed.
[in]_textSet to true = text data, false = binary data.
[in]_instrumentIdSensor/Instrument ID
