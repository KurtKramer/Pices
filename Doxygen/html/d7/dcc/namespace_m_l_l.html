Detailed Description

Namespace for all Machine Learning and SIPPER specific code.

Contains Classes that are specific to SIPPER's physical characteristics such as Instrument Data.

A library of routines used for Machine Learning, building Classifiers, etc. It also includes code that is specific to PICES and SIPPER. At some point in the future we need split this library into three libraries. MachineLearning, DataBase, and PicesRoutines.

This library covers three different areas that at some time in the future should be made into three separate libraries:

  1. Generic Machine Learning code.
  2. Database Related Code.
  3. Analysis.
  4. PICES specific code that would be built on top of the Machine Learning and Database code.

"Machine Learning Code"

"Feature Data"

There are several Feature Data formats supported. Each one has its own class that is derived from 'FeatureFileIO'. The description of the data is managed by 'FileDesc'. For each type of dataset there will exist on one instance of a FileDesc class.
