14  /**
15  *@class BitString
16  *@brief Allows you to manage very long bit strings.
17  *@author Kurt Kramer
18  *@details Useful when you need to deal with very large yes/no decisions. For example performing Feature Selection
19  * on a DNA dataset where you can have 50,000+ features. You need to keep a list of which feature combinations have
20  * been tried. You can use a BitString to do this where a particular bit indicates a particular feature. In the
21  * feature selection case you may want to track several thousand feature combinations. If you did this using arrays
22  * you would require very large amount of memory to accomplish this. With BitString's the memory requirement is
23  * reduced to 1/8'th allowing for more efficient use of memory.
24  *
25  * This class will manage Bit-Strings up to UINT_MAX in length. Logical operations such as bitwise AND, OR, and NOT
26  * are supported plus others. An example of where this class is used is in PicesLibrary MLL::FeatureNumList.
27  */
